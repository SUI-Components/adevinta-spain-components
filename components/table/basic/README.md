# TableBasic

> Table basic to show tabular information.

![Table Basic](./assets/preview.png)

## Installation

```sh
$ npm install @s-ui/sui-table-basic --save
```

## Usage

### Basic usage

```js
import TableBasic from '@s-ui/sui-table-basic'

return <TableBasic />
```

# PaginationBasic

> Component with a basic pagination system

<!-- ![](./assets/preview.png) -->

## Installation

```sh
$ npm install @s-ui/react-pagination-basic --save
```

## Usage

### Basic usage
```js
import PaginationBasic from '@s-ui/react-pagination-basic'

return (
  <PaginationBasic
    currentPage={20}
    handlePaginate={() => {}}
    showFirstLast
    totalPages={40}
    window={10}
  />
)
```

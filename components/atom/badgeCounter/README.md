# AtomBadgeCounter

> Atom Element: SUI badgeCounter

## Installation

```sh
npm install @schibstedspain/sui-atom-badge-counter --save
```

## Usage

```js
import BadgeCounter from '@schibstedspain/sui-atom-badge-counter'

render {
  return (
    <BadgeCounter />
  )
}

```
